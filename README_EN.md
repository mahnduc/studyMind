## 💡 About Uxie
**Uxie** is a powerful open-source tool designed to transform static documents like PDFs or Markdown files into interactive learning systems. The project leverages **AI Agents** to analyze and generate content directly within the web environment, empowering you to master knowledge more intelligently.

## 🚀 The Local-first Web Philosophy
Uxie bypasses traditional server-side storage for personal data, ensuring maximum user privacy:
*   **Absolute Privacy:** Your data is encrypted and stored exclusively in your browser; no third-party interaction is allowed.
*   **OPFS Performance:** Utilizes the **Origin Private File System**—a dedicated virtual file system per website—enabling high-performance data access natively in the browser.
*   **Server Independence:** The entire workflow, from storage to search, executes locally on your device.

## ✨ Key Features
- [x] **Smart Rich-text Editor:** A minimalist editing experience powered by **Tiptap**, optimized for academic note-taking and documentation.
- [x] **AI-Powered Quiz:** Automatically generate intelligent quizzes from your documents via a **RAG No-Embedding** pipeline, optimizing resources without sacrificing accuracy.
- [x] **Bento Grid Dashboard:** A modern, sleek dashboard interface featuring a vibrant "Pink & White" design aesthetic.
- [x] **Hybrid Search:** Combines **BM25** algorithms with semantic search to ensure the most precise document retrieval possible.

## 🛠 Tech Stack
*   **Framework:** Next.js & TypeScript.
*   **Styling:** Tailwind CSS with a Bento Grid layout.
*   **Editor:** Tiptap Framework.
*   **Storage:** OPFS (Origin Private File System) & Browser Native Storage.

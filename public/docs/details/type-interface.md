# Sự khác biệt giữa Type và Interface (TS)

|Feature	|interface	|type|
|:-|:-|:-|
|Purpose	|Describes the shape (structure) of an object or class.	|Creates a type alias for any type (object, union, intersection, primitive, tuple, etc.).|
|Nature	|Extensible — can be merged and extended.	|Fixed — cannot be reopened once defined.|
